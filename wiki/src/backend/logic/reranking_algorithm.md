# 하이브리드 재랭킹 알고리즘 (Hybrid Re-ranking Algorithm)

이 문서는 검색된 향수 후보군 중 최종 추천 순위를 결정하는 통합 점수 산출 공식을 정의합니다.

## 1. 최종 점수 산출 공식
최종 추천 점수는 아래 세 가지 요소의 가중 합산으로 계산됩니다.

$$Final Score = (S_{aura} \times w_1) + (S_{RAG} \times w_2) + (S_{note} \times w_3)$$

- $S_{aura}$: 아우라 벡터 유사도 (Cosine Similarity)
- $S_{RAG}$: 시맨틱 검색 유사도 (Pinecone/RAG Score)
- $S_{note}$: 사용자 선호 노트 일치도 (Matching Count)

### 1.1. 적용 가중치 (Applied Weights)
| 요소 | 가중치 | 설정값 | 비고 |
| :--- | :--- | :--- | :--- |
| 아우라 가중치 ($w_1$) | 0.5 | 50% | 수치적 취향 분석의 핵심 지표 |
| RAG 가중치 ($w_2$) | 0.5 | 50% | 이미지/자연어 기반의 감성적 유사도 |
| 노트 보너스 ($w_3$) | 0.03 | +0.03 | 매칭 노트당 가산점 (최대 0.1점으로 캡핑) |

### 1.2. 설정 근거 (Rationale)
1. **정량적 분석과 정성적 분석의 균형 (5:5)**: 사용자의 5축 아우라 분석(정량)과 VLM을 통한 이미지 무드 분석(정성)이 동일한 비중으로 반영되도록 설계되었습니다. 이는 '기술적 정밀도'와 '감성적 직관'의 균형을 유지하여 추천의 설득력을 높이기 위함입니다.
2. **명시적 선호도의 보정 (Note Bonus)**: 사용자가 직접 선택한 '선호 노트'는 강력한 구매 의사 결정 요인이므로, 유사도가 비슷한 제품들 사이에서 선호 노트를 포함한 제품이 우선 순위를 선점하도록 가산점 체계를 도입했습니다.
3. **상한선 설정 (Max 0.1)**: 특정 노트 일치 여부가 전체 추천 알고리즘의 밸런스를 무너뜨리지 않도록 가산점의 총합을 0.1로 제한하였습니다.

## 2. 요소별 상세 로직
- **아우라 유사도 ($S_{aura}$)**: 사용자의 5축 아우라 벡터와 향수의 고유 아우라 벡터 간의 거리를 측정합니다.
- **RAG 유사도 ($S_{RAG}$)**: VLM(Vision Language Model)이 생성한 쿼리와 향수의 설명/메타데이터 간의 임베딩 유사도를 Pinecone을 통해 산출합니다.
- **노트 가산점 ($S_{note}$)**: `selectedNotes`에 포함된 향료가 향수의 탑/미들/베이스 노트에 존재할 경우 가중치를 부여합니다.

## 3. 재랭킹 임계값 (Threshold)
- 최소 유사도 점수 미달 시 추천 목록에서 제외하거나 대체 상품을 탐색합니다.
